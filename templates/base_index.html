All Ghiro features can be controlled via web interface. You can upload images, bunch of images, navigate reports, get a quick or deep overview of images analysis.

Image analysis

Photo analysis with many techniques: Error Level Analysis (ELA), MIME information, GPS localization, thumbnail extraction and thumbnail consistency check.

Metadata extraction

Metadata are divided in several categories depending on the standard they come from. Image metadata are extracted and categorized. For example: EXIF, IPTC, XMP.

Automation

Ghiro is developed to handle tons of images, you can group them in cases, assign users to cases with proper permission, search images for properties or image location.
